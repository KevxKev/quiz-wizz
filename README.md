# olympus-night-integrated

Clean integration project: Olympus Night visual direction + Quiz Wizz game engine.

## First implemented flow

- Host creates room
- Phone joins room
- Joined player appears in host lobby in realtime

## Run

1. Install deps:

   npm install

2. Configure env:

   copy .env.local.example to .env.local and fill Supabase values

3. Start dev server:

   npm run dev

4. Open:

   - Host: http://localhost:3002/host
   - Phone join: http://localhost:3002/join
